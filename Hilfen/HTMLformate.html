Bootstrap 5 Mini-Spickzettel

Die 3 wichtigsten Regeln für dein Game-Layout

1. Farben & Bedeutung

Ersetze primary durch das Stichwort deiner Wahl. Gilt für Buttons (btn-), Hintergründe (bg-) und Text (text-).

Stichwort Farbe Bedeutung / Nutzung Beispiel (Button)
primary Blau Hauptaktion, Speichern, Weiter
success Grün Erfolg, Kaufen, Erledigt
danger Rot Löschen, Fehler, Abbrechen
warning Gelb Warnung, Gold, Wichtig
secondary Grau Unwichtiges, Zurück, Info
dark Schwarz Dunkles Design, Footer

2. Abstände (Margin & Padding)

Aufbau: [Art][Richtung]-[Größe] (z.B. mt-3)

Die Buchstaben

  • m = Margin (Außenabstand)
  • p = Padding (Innenabstand)
  • t = Top (Oben)
  • b = Bottom (Unten)
  • s = Start (Links)
  • e = End (Rechts)
  • x = Links & Rechts
  • y = Oben & Unten

Die Größen (0 bis 5)

Größe 0 (0px)
Größe 1 (klein)
Größe 3 (mittel - Standard)
Größe 5 (riesig)

3. Das Raster (Layout)

Eine Zeile (row) hat immer 12 Slots.

4. Wichtige Helfer

d-none Element komplett verstecken (Display: none).
d-flex Macht eine Box zur Flexbox (für Ausrichtung).
gap-2 Erzeugt Lücken zwischen Flex-Elementen.
w-100 Breite auf 100% zwingen.
text-center Text zentrieren.
fw-bold Text fett machen.
